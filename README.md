## 🚀 How to Use

### Local Development
Simply open `index.html` in a web browser. No build process required.

### Deployment
1. Upload all files in the `landing2025` directory to your web server
2. Ensure the parent directory structure is maintained for image/logo paths
3. Update any absolute URLs if needed

### Customization

#### Update Content
- Edit text directly in `index.html`
- Modify colors in `style.css` using CSS variables (`:root` section)

#### Update Images
- Replace images in the parent `images` directory
- Update image paths in `index.html` if needed

#### Update Styling
- All styles are in `style.css`
- Uses CSS variables for easy theming
- Organized by section with clear comments
